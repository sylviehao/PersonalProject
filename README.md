# Table Gamer - 桌遊人
#### A guiding app for board game lovers to share posts, get practical tools and start games together

- 貼文分享 - 洞悉時下高人氣桌遊
- 揪團功能 - 尋找一起遊玩的夥伴
- 遊戲編輯 - 推廣你最喜歡的遊戲
- 收藏喜好 - 聚會挑選時不再迷惘
- 實用助手 - 小工具增加遊戲體驗
- 個人主頁 - 紀錄你的遊戲里程碑
